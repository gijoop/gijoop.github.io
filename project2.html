Chińskie szachy online

Projekt aplikacji desktopowej do gry w chińskie szachy online, umożliwiający grę z innymi użytkownikami oraz z komputerem. Aplikacja została zaimplementowana w języku Java z wykorzystaniem biblioteki JavaFX.

Możliwości aplikacji:

  • Możliwość gry z innymi użytkownikami
  • Możliwość gry z botem
  • Możliwość zapisywania i wczytywania gier z bazy danych
  • Dwa warianty rozgrywki: standardowy i "Fast-Paced"

Wykorzystane technologie

  • Java dla logiki aplikacji
  • JavaFX dla interfejsu graficznego klienta
  • Spring do komunikacji z bazą danych
  • Jackson do serializacji obiektów po sieci
  • JUnit do testów jednostkowych

Przykładowy kod - kalkulacja dostępnych ruchów w wariancie przyspieszonym (Fast-Paced)

@Override public ArrayList<Position> getReachableJumps(Position position) { ArrayList<Position> jumps = new ArrayList<>(); int x = position.getX(); int y = position.getY(); int maxDistance = board.getSize() + 1; for (Position dir : DIRECTIONS) { Position firstPawn = null; int pawnDist = 1; for(int dist = 1; dist <= maxDistance; dist++){ Position potentialPawn = new Position(x + dist * dir.getX(), y + dist * dir.getY()); if(board.isOccupied(potentialPawn)){ firstPawn = potentialPawn; pawnDist = dist; break; } } if(firstPawn == null){ continue; } boolean isJumpable = true; for(int behind = 1; behind <= pawnDist; behind++){ Position potentialPawn = new Position(firstPawn.getX() + behind * dir.getX(), firstPawn.getY() + behind * dir.getY()); if(board.isOccupied(potentialPawn)){ isJumpable = false; break; } } if(isJumpable){ jumps.add(new Position(x + 2 * pawnDist * dir.getX(), y + 2 * pawnDist * dir.getY())); } } return jumps; }
